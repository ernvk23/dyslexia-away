DyslexiaAway
About Get

Read the web, your way.

A minimal browser extension that applies a dyslexic‑friendly font to websites. In case reading has ever felt hard, this might help. Free forever:)

Get for Chrome Get for Firefox

What it does

Applies OpenDyslexic font

Changes website fonts to OpenDyslexic.

Adjustable spacing

Increase letter and line spacing.

Site exclusion

Skip pages where the font change could cause issues.

Light & dark themes

Matches your system theme or choose manually.

Cross‑browser compatible

Works on Chrome/Firefox compatible browsers (including Firefox for Android).

Multi‑language interface

Available in English, Spanish, French, German, Italian, Portuguese, and Russian.

No data collection

Everything happens locally. No tracking.
