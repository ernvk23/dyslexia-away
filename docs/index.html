DyslexiaAway
Get About

Read the web, your way.

A simple browser extension that applies a dyslexic‑friendly font. If reading has ever felt hard, this might help. Free forever :)

Get for Chrome Get for Firefox

What it does

Applies dyslexic‑friendly fonts

Changes website fonts to Andika, Lexend, OpenDyslexic, Shantell Sans, Balsamiq Sans, or Atkinson Hyperlegible.

Some fonts may not support specific languages (e.g., Cyrillic/Greek).

Adjustable spacing

Increase letter and line spacing.

Site exclusion

Skip pages where the font change could cause issues.

Light & dark themes

Matches your system theme or choose manually.

Cross‑browser compatible

Works on Chrome, Firefox, Edge, Brave, and Firefox for Android.

Multi‑language interface

Available in English, Spanish, French, German, Italian, Portuguese, and Russian.

No data collection

Everything happens locally. No tracking.
